VOLUME RAY CASTING

HOW THIS WAS RAN

Locally

First, I had to clone the GitHub repository to my computer using:

git clone https://github.com/aruvham/javascript-ray-casting.git

Next, the project was based on express.js which a node.js framework. So I had to install node.js on my computer and run the commands below on my command line to start the project.

cd javascript-ray-casting
node index.js

Now, this starts the a server for the project on Localhost port 3000.
So I had to visit http://localhost:3000 in my browser to see the output of the project.

Remotely On This Page

With the project based on the P5.js library, the ray casting work was implemented using using JavaScript. The resulting JavaScript code was then injected into the body of the and index.html file, which enables the project to run perfectly in any JavaScript supported browser.

INTERACTING WITH THE WORK

Using the arrow keys, you can interact with the project to move the source of the ray around and change the direction of the cast rays.

  • The Left & Right arrows change the direction of the cast rays.
  • The Top & Down arrows move the source of the rays forward and backwards.
